## 🎯 Sobre o Projeto

O objetivo deste site é servir como um currículo interativo e uma vitrine para minhas habilidades no desenvolvimento de interfaces de usuário (UI), experiência do usuário (UX) e consumo de APIs.

### ✨ Funcionalidades
* **Apresentação Pessoal:** Um resumo sobre mim, minha formação e meus interesses no mundo da tecnologia.
* **Vitrine de Projetos:** Seção dedicada aos projetos desenvolvidos, com links para os repositórios e descrições técnicas.
* **Habilidades e Stacks:** Exibição visual das tecnologias que domino e utilizo no dia a dia.
* **Design Responsivo:** Layout adaptável para dispositivos móveis, tablets e desktops.
* **Integração de Contato:** Formulário ou links diretos para minhas redes (LinkedIn, GitHub, E-mail).
